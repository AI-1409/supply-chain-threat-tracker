SC-2024-003 · RubyGems

MEDIUM (6.8)medium
Description

A coordinated campaign involving 760+ malicious RubyGems packages targeting typosqats of popular gems. These packages contained malicious code that hijacked clipboard content to steal cryptocurrency wallet addresses during paste operations, substituting victim addresses with attacker-controlled addresses.
